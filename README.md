I have organized my learning journey into distinct modules, each focusing on a specific technical milestone:

* **03 | Logic & Decision Trees:** Implementation of nested conditional logic (`if/elif/else`) to build complex, branching decision paths.
* **04 | Stochastic Elements:** Utilizing the `random` library to introduce probability-based outcomes and list-indexing.
* **05 | Iterative Sequences:** Mastery of `for` and `while` loops to handle randomization and data entropy.
* **06 | Functional Programming:** Developing reusable, modular functions to solve autonomous navigation problems.
